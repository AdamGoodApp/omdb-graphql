GraphQL interface for the [Open Movie Database API](http://www.omdbapi.com/).

You'll need an api key from the above site.

1. Check out the project.
2. `npm install`
3. `node index.js`
4. Make GraphQL queries to `http://localhost:3000/graphql`.
